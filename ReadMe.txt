Please read carefully following installation instructions for the Sinhala Locale,Sinhala IME and for sinhala Phonetic Translater

Fonts
======
To run following applications correctly you will need KaputaUnicode and KandyUnicode fonts which comes with this bundle or can be downloaded from http://www.info.lk/slword/swdowns.htm



Installing the Sinhala Locale and execute the Sinhala Locale Example
=====================================================================

1.  Navigate to the java_sinhala_localization\SinhalaLocale\dist folder


2.  Copy the SinhalaIME.jar file to the Java VM extension folder (i.e.C:\Program Files\Java\jre1.6.0\lib\ext in Windows) or Java class path.

3   Navigate to the java_sinhala_localization\SinhalaLocaleExamples\dist folder


4   Double click the SinhalaLocaleExamples.jar file or 

    change the cmd prompt to the relevant directory and type java -jar SinhalaLocaleExamples.jar to execute the application 




Installing the Sinhala IME in java Vitual Machine
==============================================================

1.  Navigate to the java_sinhala_localization\SinhalaIME\dist folder

2.  Copy the SinhalaIME.jar file to the Java VM extension folder (i.e.C:\Program Files\Java\jre1.6.0\lib\ext in Windows) or Java class path.



Executing Sinhala Phonetic Translator
===========================================

1.   Navigate to the java_sinhala_localization\SinhalaPhoneticTranslator\dist folder

2.   Double click the SinhalaPhoneticTranslator.jar file or 
    
Change the cmd prompt to the relevant directory and type java -jar SinhalaPhoneticTranslator.jar to execute the     application on the command prompt.    

Executing Java Sinhala Prototype
=======================================
          

1. Navigate to the java_sinhala_localization\JavaSinhalaPrototype\dist folder

2  Double click the JavaSinhalaPrototype.jar file or 
    
Change the cmd prompt to the relevant directory and type java -jar JavaSinhalaPrototype.jar to execute the application on the command prompt.
